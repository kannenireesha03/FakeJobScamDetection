 Fake Job Scam Detection System
 Project Overview

The **Fake Job Scam Detection System** is a Machine Learning-based application that detects fraudulent job postings using Natural Language Processing (NLP) and classification algorithms.

The system analyzes job descriptions and predicts whether a job post is **Real ** or **Fake **, helping job seekers avoid online recruitment scams.

---

##  Objective

- Detect fake job postings automatically
- Protect users from online job scams
- Apply NLP and Machine Learning techniques for text classification
- Provide real-time prediction through a simple interface

---

##  Tech Stack

- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-learn
- **NLP:** TF-IDF Vectorization
- **Model:** Logistic Regression / Naive Bayes
- **Frontend (Optional):** HTML, CSS
- **Deployment (Optional):** Streamlit / Flask
